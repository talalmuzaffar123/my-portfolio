Sentiment Anaysis of Memes

Data Analysis, Machine Learning, Ensemblers

One of the projects I worked on was developing a sentiment analysis model for memes. We used various machine learning algorithms and natural language processing techniques to train the model to recognize the sentiment behind the textual content of memes. The goal of this project was to create a tool that could help individuals and businesses better understand how their content is being received by their audience. We also explored different ways to visualize the results of the analysis, including sentiment heatmaps and word clouds. The project was a great opportunity to work with cutting-edge technology and apply it to a fun and unique application.

Web Application for Finding Similar Songs using LSH

Data Analysis, Min Hashing, Locality Sensitive Hashing, Flask

I worked on a project that developed a Flask application using Min Hashing and Locality Sensitive Hashing algorithms for audio analysis. We created a tool that can compare an uploaded audio file to a pre-existing dataset to identify similar audio files. Our findings suggest that the application has potential applications in the music industry, such as helping with music recommendations and suggesting similar songs. The project allowed us to explore cutting-edge technology and develop practical applications for it.
